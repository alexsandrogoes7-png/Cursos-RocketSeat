## O peso das cores

- Nessa aula, o professor revelou o "segredo dos designers" para nunca mais errar na mão com as cores: a Regra 60-30-10. 
    - 60% (Cor Dominante)
    - 30% (Cor Secundária)
    - 10% (Cor de Destaque/Sotaque)

    
- É uma técnica matemática que garante equilíbrio visual, evitando que o site pareça um "carnaval" ou fique cansativo para os olhos.

- Aprendemos que a harmonia não é sorte, é proporção. 

- O foco não é apenas beleza, mas Acessibilidade (garantir que o texto seja legível sobre o fundo) e Experiência do Usuário.
