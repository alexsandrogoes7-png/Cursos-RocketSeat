## Espaço não é dinheiro

- Essa aula trouxe uma frase que é um verdadeiro mantra no design moderno: "Espaço não é dinheiro". 

- O professor desmistificou a ideia de que cada pixel vazio da tela é um desperdício de espaço que deveria estar preenchido com anúncios, botões ou textos.

- Na verdade, o espaço branco é um dos elementos mais luxuosos de uma interface. 

- Ele é o que separa um site que parece um "panfleto de supermercado" de uma plataforma de alta tecnologia e confiança.
