## Escalando textos

- Escalar texto não é escolher tamanhos aleatórios; é criar uma escada onde cada degrau faz sentido. 

- Se o título é gigante e o texto é minúsculo, o visual quebra. Se são quase iguais, o usuário não sabe o que ler primeiro.

- A ideia é usar uma proporção matemática para que o tamanho do H1, H2 e do corpo de texto tenham uma relação harmônica. 

- Isso cria um ritmo visual que guia o olho e deixa a leitura confortável em qualquer tela.
