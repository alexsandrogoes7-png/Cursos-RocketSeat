## Criando composições

- O foco desta aula é como construir uma seção de alto impacto visual do zero, usando formas e luz para valorizar o produto.

- Pontos principais:

    - Composição com Cores: Aplicação da regra 60/30/10, usando o roxo (30) como cor de fundo para criar contraste e destacar o motor.

    - Técnica de Iluminação: Uso de elipses com Layer Blur para criar pontos de luz e profundidade, dando um aspecto tridimensional à peça.

    - Edição de Formas: Criação de molduras personalizadas a partir de retângulos, usando a edição de pontos para "baular" (arredondar) cantos e criar curvas exclusivas.

    - Uso de Assets: Importação do motor como um componente pronto do Style Guide, garantindo agilidade e consistência.

    - Efeito Backlight: Aplicação de luz branca por trás do objeto para simular que ele está flutuando e dar foco total ao item técnico.
