## Selecionando imagens

- O professor mostra como transformar fotos simples em composições profissionais e criar variações de produtos sem sair do Figma.

- Pontos principais:

    - <strong Composição do Topo:> Uso de fotos urbanas invertidas com máscaras de gradiente linear para criar profundidade e legibilidade para os textos.

    - <strong Edição Não-Destrutiva:> Diferente do Photoshop, o Figma esconde partes da imagem via máscara, permitindo ajustes constantes sem perder o arquivo original.

    - <strong Variações de Produto:> Partindo de uma única foto, o professor criou 3 modelos de caminhão (P200, M600, G900) apenas alterando rodas, caçambas e para-choques com a Pen Tool.

    - <strong Efeitos de Realismo:> Aplicação de Layer Blur e opacidade para simular movimento (ex: volante escamoteável) e ajustes de contraste para integrar o caminhão ao fundo.

    - <strong Perspectiva:> A importância de alinhar o objeto com a linha do horizonte da foto de fundo para manter o realismo.
