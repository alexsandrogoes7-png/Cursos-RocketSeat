# Refinamentos nos textos

## 1. Tipografia (Text Styles)
* **Criação de Classes:** Transformar textos soltos em estilos globais (Menu, Banner, Botões, Conteúdo).
* **Hierarquia Visual:** Padronização de títulos em **48px** e **32px** para manter a consistência em todas as páginas.
* **Line Height:** Uso de **120%** ou **150%** para garantir legibilidade e respiro.
* **Variações Mixed:** Separação de estilos *Bold* e *Regular* dentro de categorias (ex: `Banner/Subtítulo`).

## 2. Paleta de Cores (Color Styles)
* **Escala Cromática:** Organização de cores por níveis (ex: `Roxo/400`, `600`, `700`).
* **Substituição do Preto:** Uso de **Cinza Escuro** em vez de preto puro (#000000) para reduzir o cansaço visual.
* **Amarelo de Destaque:** Criação de estilo específico para botões de conversão (CTA).

## 3. Organização e Workflow
* **Nomenclatura:** Uso de barras (ex: `Conteúdo/Bold`) para criar pastas automáticas no Figma.
* **Manutenção Ágil:** A alteração de um único estilo global atualiza automaticamente todo o layout, facilitando o retrabalho.
