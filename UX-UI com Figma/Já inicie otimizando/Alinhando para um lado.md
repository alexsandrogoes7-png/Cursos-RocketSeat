## Alinhando para um lado

- O professor explica que os Constraints são fundamentais para definir como os objetos se comportam e se alinham quando a tela ou o frame são redimensionados. 

- O instrutor recomenda verificar se essa configuração está ativa ao adicionar itens à página, evitando erros de posicionamento. 

- A utilização correta automatiza ajustes em componentes e animações, otimizando o fluxo de trabalho. 

- Além disso, destaca-se a importância de prever como o texto se comportará ao crescer dentro de um elemento para manter a integridade do design.
