## Configurando os Grids

- O professor demonstra como configurar e transformar as grades de layout em estilos reutilizáveis para manter a consistência entre Desktop e Mobile. O objetivo é garantir que todo o conteúdo siga uma simetria rigorosa dentro do projeto Virtus Motors.


- Pontos principais:

    - Configuração Desktop: 12 colunas, margem de 100px e gutter (espaçamento) de 20px.

    - Configuração Mobile: 4 colunas, margem de 24px e gutter de 16px.

    - Transformação em Estilo: Criação de estilos globais nomeados (ex: "Desktop 1440" e "Mobile 360") para facilitar o acesso.

    - Organização no Style Guide: Manutenção dos frames de grid em uma área dedicada para consulta.

    - Aplicação Prática: Substituição de grades manuais pelo estilo criado diretamente na interface do projeto.

    - Consistência Visual: Alinhamento de todos os elementos (textos, imagens e botões) estritamente dentro das colunas da grid.
