## Apresentando o projeto

- O professor apresenta o fluxo de trabalho profissional para a criação da landing page do caminhão Cargo P200. O foco é organizar o projeto para evitar bloqueios criativos e facilitar a entrega.

- Pontos principais:

 - Projeto Prático: Criação de site para a marca fictícia Virtus Motors.

 - Organização Lateral: Divisão por Briefing, Pesquisa, Wireframe e Interface.

 - Benchmarking: Análise visual de concorrentes (Kia, Mercedes, Scania).

 - Moodboard: Centralização de referências no Figma para alinhar com o cliente.

 - Ativos Visuais: Curadoria de fotos técnicas (motor e interior) e tratamento da imagem principal.

 - Metodologia: Estudo por replicação e melhoria do arquivo original fornecido.
