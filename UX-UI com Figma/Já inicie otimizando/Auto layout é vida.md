## Auto layout é vida

- O professor apresenta o Auto Layout como uma ferramenta essencial para criar designs adaptativos e responsivos. 

- A recomendação é aplicá-lo após a definição da composição visual, utilizando ajustes como fill container, hug contents (citado como fit) e fixed para controlar como os elementos se comportam em diferentes telas. 

- O professor enfatiza a importância de dominar as configurações de altura, largura e o entendimento de breakpoints para garantir que o layout seja funcional em diversos dispositivos, sempre priorizando a experiência do usuário.
