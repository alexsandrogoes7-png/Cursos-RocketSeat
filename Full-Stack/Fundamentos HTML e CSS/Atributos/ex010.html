Atributos globais

O professor explica que os atributos globais são comuns a qualquer elemento HTML, independentemente da tag utilizada. Diferente de outros atributos, eles não alteram a estrutura fundamental do elemento, mas cumprem papéis essenciais, como promover a acessibilidade (atributos aria-) ou criar eventos na página. O instrutor enfatiza que não é necessário memorizar toda a lista de uma vez; o ideal é focar nos atributos fundamentais e aprender os demais conforme a necessidade e a demanda prática dos projetos.

Atributos Usados

Id

O professor explica que o atributo id funciona como um "RG" do elemento, ou seja, um identificador único dentro de uma mesma página. Ele é utilizado para permitir que o CSS ou o JavaScript acessem e manipulem um elemento específico. O instrutor reforça regras importantes de escrita: o nome do ID não deve começar com números, não deve conter caracteres especiais nem espaços. Caso seja necessário separar palavras, a recomendação é utilizar um traço. Por fim, enfatiza-se que o valor de um ID jamais deve ser repetido em outros elementos do mesmo documento.
