#  O que é CSS?

**CSS** (*Cascading Style Sheets* ou **Folha de Estilo em Cascata**) é a linguagem usada para definir a aparência das páginas web (HTML).

###  Conceitos Chave
* **Função:** Modifica cores, fontes, posicionamento e até cria animações.
* **Estrutura Base:** Funciona através de **Propriedade** e **Valor**.
    * Exemplo: `background-color: blue;`
* **A "Cascata":** O nome vem do fato de que os estilos são aplicados em sequência (empilhados). Você define uma regra após a outra para construir o visual final.
