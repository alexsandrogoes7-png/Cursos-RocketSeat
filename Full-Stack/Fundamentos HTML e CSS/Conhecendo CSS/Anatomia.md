# Anatomia da Escrita CSS

O CSS funciona através de **Declarações** que dizem ao navegador como estilizar um elemento HTML específico.


# Exemplo:

h1 {
  color: blue;                 /* Cor do texto */
  font-size: 60px;             /* Tamanho da fonte */
  letter-spacing: 2px;         /* Espaçamento entre letras */
  text-transform: uppercase;   /* Transforma em CAIXA ALTA */
}
