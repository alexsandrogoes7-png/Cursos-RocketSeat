Anatomia das Tags

Nesta aula, o professor desvendou como o HTML funciona por "baixo do capô". A anatomia de uma tag é a base de tudo: você tem a tag de abertura (o "abre alas"), o conteúdo no meio e a tag de fechamento (com a barra /) para dizer onde aquilo termina. Tudo junto forma o que chamamos de elemento. Aprendemos também sobre os atributos, que são as configurações extras (tipo o src de uma imagem), e as tags vazias (ou self-closing), como a
(quebra de linha) e a , que não precisam de fechamento porque não "guardam" nada dentro, elas já são o próprio conteúdo.
