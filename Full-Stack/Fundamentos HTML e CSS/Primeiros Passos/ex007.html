O professor trouxe um ponto crucial: o HTML é literal. Se você digita < no meio de um texto, o navegador acha que você está abrindo uma tag e se perde todo. Por isso, usamos o que chamamos de Entidades HTML. É como um "código secreto" para dizer ao navegador: "Ei, não interprete isso como código, apenas desenhe o símbolo na tela". Além disso, vimos a tag , que é perfeita para mostrar trechos de programação, pois ela usa aquela fonte de "letra de máquina" (monoespaçada) que facilita a leitura técnica.
