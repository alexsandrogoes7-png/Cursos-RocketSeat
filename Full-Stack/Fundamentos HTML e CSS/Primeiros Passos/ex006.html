Aninhamento de Tags

O professor mostrou que o HTML funciona como uma família. Quando você coloca uma tag dentro da outra (aninhar), você cria uma hierarquia: quem está por fora é o pai e quem está por dentro é o filho. É como uma caixa dentro de outra caixa. O segredo para não dar erro no código é a ordem: a última tag que você abriu precisa ser a primeira a ser fechada. Se você abrir o parágrafo e depois o itálico, tem que fechar o itálico primeiro para "trancar" o conteúdo direitinho.

Por que isso importa? Organização: Se você aninha errado, o navegador pode se confundir e aplicar estilos onde não devia.
