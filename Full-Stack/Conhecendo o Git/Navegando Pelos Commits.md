## Navegando Pelos Commits

- Nessa aula o professor mostrou como o git log funciona como a nossa "máquina do tempo" para ver o histórico do projeto. 

- Aprendemos que cada commit tem um código único (ID) e que, com o git checkout, conseguimos viajar para qualquer versão antiga para conferir o que foi feito ou até criar uma nova branch para testar ideias.
