## Recuperando Um Arquivo Deletado

- Nessa aula o professor ensinou que o Git é o nosso "seguro contra acidentes": se a gente deletar um arquivo sem querer, ele não some para sempre. 

- Aprendemos a usar o git checkout para buscar o arquivo direto da última versão salva e como limpar a "fila de espera" (stage area) se a gente mudar de ideia sobre uma exclusão.
