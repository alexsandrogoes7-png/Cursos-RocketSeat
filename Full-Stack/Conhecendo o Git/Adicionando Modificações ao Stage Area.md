## Adicionando Modificações ao Stage Area

- Nessa aula o professor explicou como o git status é o nosso "olho" para ver o que mudou no projeto.

- Aprendemos a usar o git add . para colocar tudo na fila de espera (Stage Area) e que, se a gente se arrepender, o git rm --cached tira o arquivo dessa fila antes do commit final.
