## Comandos Para Atualizar e Enviar Atualizações Entre Repositórios Remoto e Local

- Nessa aula o professor ensinou que o Git Pull é o comando para "puxar" as novidades que outras pessoas fizeram no GitHub para o seu computador. 


- Aprendemos que a regra de ouro é sempre dar um pull antes do push, para evitar que o Git se confunda e gere conflitos entre as versões.
