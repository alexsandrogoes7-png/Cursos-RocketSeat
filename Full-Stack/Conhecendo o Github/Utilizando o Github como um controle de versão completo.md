## Utilizando o Github como um controle de versão completo

- Nessa aula o professor mostrou que o GitHub não é só um lugar para guardar código, mas um editor completo nas nuvens.

-  Aprendemos que, ao apertar a tecla . (ponto) no teclado enquanto estiver em um repositório, o GitHub abre uma versão do VS Code direto no navegador. 

- Lá você consegue editar, apagar arquivos e fazer commits sem precisar abrir o terminal ou ter o Git instalado no PC.
