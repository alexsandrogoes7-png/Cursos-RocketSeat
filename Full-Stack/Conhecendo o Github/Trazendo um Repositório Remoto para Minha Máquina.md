## Trazendo um Repositório Remoto para Minha Máquina

- Nessa aula o professor ensinou o comando git clone, que é como "baixar" um projeto inteiro do GitHub para o seu computador de um jeito inteligente. 

- Aprendemos que basta copiar o endereço do repositório e usar o terminal para criar uma cópia exata na sua máquina, permitindo que você comece a trabalhar no código na hora, mesmo que o nome da pasta no seu PC seja diferente do nome no site.
